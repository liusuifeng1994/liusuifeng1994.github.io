使用范围

  1. 顺序存储结构， 通常是数组；
  2. 有序性，待查找的数组有序或者部分有序
  3. 要求时间复杂度低于O(n)，或者直接要求时间复杂度为O(logn)

基本类型

  1. 在数组中查找符合条件的元素的索引
  2. 在一个有上下界的区间里搜索一个整数
  3. 判别条件是一个函数

基本思路

二分法的思路与模板较多，本文主要介绍基于“排除法”写的二分查找算法

  1. 区间的索引范围是 [0, size)
  2. while (left < right)
    没有=, 退出条件:left = right. 注意：left(right)位置的值可能程序还没有读取到，因此“有可能”需要再对left(right)这个位置的值是否是目标元素的值做一次判断。
  3. if else, 思考nums[mid]满足什么性质的时候，mid不是解
    往往思考什么时候不是解相对比较好想。
  4. 中位数判断，以免死循环
    mid在右边是闭合,即right=mid：int mid = left + (right - left) / 2
    mid在左边是闭合,即left=mid：int mid = left + (right - left + 1) / 2
  5. 退出最后检验, “后处理”
    退出时, 一定有left == right成立。有时可以直接返回left(right)或者与left相关的数值，有时还须要再做一次判断，判断left 与 right是否是我们需要查找的元素.
